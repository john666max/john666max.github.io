# John Chen (陈子皓) - Portfolio Website

A modern, Apple-inspired portfolio website built with HTML, CSS, and JavaScript. Designed to showcase skills, experience, and projects for university and job applications.

## Features

- **Dark Theme**: Sleek, modern dark interface with careful attention to typography and spacing
- **Smooth Scrolling**: Fluid scroll animations and parallax effects throughout the site
- **Responsive Design**: Fully responsive and mobile-optimized
- **Apple-Inspired**: Clean design language inspired by Apple's website aesthetic
- **Performance Optimized**: Fast loading with smooth 60fps animations

## Sections

1. **Navigation** - Fixed header with smooth scroll links
2. **Hero** - Eye-catching introduction with call-to-action buttons
3. **About** - Personal introduction and key highlights
4. **Experience** - Detailed experience with FRC5409 and technical skills
5. **Skills** - Organized skill categories with interactive tags
6. **Contact** - Multiple ways to get in touch
7. **Footer** - Simple footer with copyright

## Technologies Used

- HTML5
- CSS3 (with animations and transitions)
- Vanilla JavaScript
- System fonts (Apple San Francisco)

## Getting Started

1. Clone this repository
2. Open `index.html` in your browser
3. The site will be live at `https://john666max.github.io`

## Customization

All content is easily customizable:
- Edit text in `index.html`
- Modify colors in `styles.css` (CSS variables in `:root`)
- Add animations in `script.js`
